Commands:

  • /menu

    Opens the General Command Panel

  • /admin

    Opens the Admin Panel

  • /purge

    Purges a channel.

  • /staff-app

    This allows staff to create a simple ticket system for staff applications.

  • /report

    This command is used to report users directly to Server Staff!

  • /bug

    Use this to report a bug!

  • /ping

    Responds with the bot's Ping!

  • Server Specific

    /invite - Use this to get a direct link to invite the bot!
